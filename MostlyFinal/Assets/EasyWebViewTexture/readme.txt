EasyWebViewTexture For Android

Note: Multimedia, such as Youtube or Video does not work.
Editor On the Android Platform API can not be used ....
Only works with Android devices.


This plug-in has been using a WebView Class of Android.

It is a plug-in that will process the screen of WebView of Android to the texture-like.

Touch operation is also available. (Note: If you use a touch of many objects, there is a possibility that a decrease in system performance occurs.)

Contains Android Java Source. (If necessary, you can use to modify.)
